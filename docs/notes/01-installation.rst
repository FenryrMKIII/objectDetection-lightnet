Installation
============

First install PyTorch and Torchvision. |br|
You can then install this library by running the following command:

.. code-block:: bash

   # Optionally install brambox (needed for training)
   pip install brambox
   pip install lightnet
   
.. Note::
   This project is python 3.6 and higher so on some systems you might want to use `pip3.6` instead of `pip`.
